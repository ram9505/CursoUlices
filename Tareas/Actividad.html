Un artículo de ejemplo

Este artículo demuestra el uso de varias etiquetas HTML para dar formato a un texto.

Destacando partes importantes

Podemos usar la etiqueta para marcar palabras clave: HTML, CSS y JavaScript son los pilares de la web.

Para enfatizar una palabra o frase, utilizamos : Este es un ejemplo de énfasis.

Para indicar texto importante, usamos : Este es un texto muy importante.

Incluyendo información adicional

Podemos agregar la fecha de publicación con la etiqueta "time": Este artículo fue publicado el 6 de abril de 2024.

Para incluir información de contacto, utilizamos "address"
